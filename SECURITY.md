# Security Policy

## Supported Versions

This project is a static frontend application. The latest version on the `main` branch is the only supported version.

## Reporting a Vulnerability

If you discover a security issue, please report it responsibly.

- Do NOT open a public issue for security vulnerabilities
- Email: adonis-jimenez@outlook.com
- Include clear reproduction steps and impact assessment if possible

Reports will be reviewed and addressed in a timely manner.

## Scope

This project does not include a backend or authentication system. Reportable issues include:
- Client-side security flaws
- Dependency vulnerabilities
- Exposure of sensitive data in the repository

Thank you for helping keep this project secure.
